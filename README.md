# Bioloptimizer

## Overview

The goal of this project is to achieve better runtime than the Biopython Python package.

## Setup

Create a new virtual environment called ".env" and activate the environment.

Install the pip packages in the requirements.txt file.

## Running Instructions

Use ```make clean``` to clean build files, and ```make``` to build the necessary packages.

Use ```python3 -B -m bioloptimizer``` as a way to run the __main__.py file.
